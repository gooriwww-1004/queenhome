개요

QMS UNIT-01은 양자 코어 기반 생활형 AI 동반자로, 일상 속에서 사람 곁에서 함께 생활하는 차세대 AI 로봇입니다. 가사 보조·일정 관리·감성 케어·생산성 지원을 하나의 신체에 통합합니다. Bio-Silicone 2.0 피부, Quantum-Photon 회로, CNT 근육 구동계의 3중 아키텍처로 설계됩니다.

2026년 개념 설계 → 2030년 프로토타입 → 2035년 상용화를 목표로 합니다.

주요 기술 스택
  • Bio-Silicone 2.0 Skin 부드럽고 내구성 강한 인공 피부. 관절 부위를 자연스럽게 감싸며 외부 충격 흡수. 실리콘 기반 2세대 소재로 촉각 센서 내장.
  • Quantum-Photon Circuits 광자 기반 초고속 에너지 전달 회로. 기존 전자 회로 대비 발열 95% 감소. 지연 시간 0에 수렴하는 실시간 제어 가능.
  • CNT Actuators 탄소 나노튜브 기반 근육 구동 시스템. 인간 근육 대비 5배 빠르고 3배 강한 힘. 유체처럼 자연스러운 동작 구현.
  • Neural Interface Core 자율 학습 + 외부 AI 연동. QMS 팀과 실시간 데이터 공유. 가정·사무·의료 보조 현장 적용.

Queen Five AirTaxi

2인승 도시 단거리 eVTOL. 원통형 캐빈 + 덕티드 팬 + 파노라믹 창. "야, 이거 진짜 편하네" 느낌의 2030년 컨셉.

DPS-AirTaxi · 민간용 2인승 eVTOL 설계 보고서
CONCEPT 2030 80–120 km/h 100km RANGE
개요

DPS-AirTaxi는 2명 탑승 가능한 도시 단거리 이동 수단. 직접 이착륙(VTOL) 방식으로 C-130 투하 없이 독립 운용됩니다. 요즘 eVTOL 트렌드(덕티드 팬 + 원통형 캐빈)를 그대로 따르면서, QMS 특유의 세련된 디자인을 입혔습니다.

속도 80~120km/h, 항속거리 100km 이내. 2030년 상용화 목표 (프로토타입 비행 2028년 예정).

주요 설계 특징
  • 본체 길이 4m 원통형 카본 파이버. 매트 그레이 톤 + 부드러운 곡선 베벨 엣지. 벽돌처럼 투박하지 않게, 세련된 마감.
  • 파노라믹 창 양쪽 사이드 전체 대형 창 — 탑승자 시야 180도 확보. 2인석, 조종석·승객석 구분 없이 심플하게.
  • 덕티드 팬 추진계 양쪽 사이드 각 1개(총 2개). 프로펠러 완전 내장 → 소음 55dB 이하, 충돌 위험 90% 감소. 공기 입구에서 흡입 → 제트처럼 후방 배출.
  • 착지 시스템 폴딩 랜딩 기어 4개. AI 자율 제어 부드러운 착륙. 도시 옥상·주차장 어디든 가능.
  • 에너지 배터리 150kWh, 30분 내 80% 충전 (2030년 기준 리튬-황 배터리 기준). Starlink GPS 연동 자율 비행.
기술 레퍼런스
  • 덕티드 팬 Lilium Jet (독일) + Archer Midnight (미국) — 공기 효율 85% 이상 덕트 방식.
  • 자율 착지 DJI 드론 + SpaceX Starlink GPS 연동 발전형.
  • 창 디자인 Boeing eVTOL 컨셉처럼 파노라믹 글라스 — 탑승자 경험 최우선.

Defense Power System · Civilian Protection

DPS 드론 · 도시 방어 시스템

방어 전용 음파 드론. 미사일·드론 요격, 도시 공중 안전 보호 목적. 공격 기능 없음. 2030년 프로토타입 개념.

DPS 드론 · 도시 방어 전용 음파 시스템 보고서
2030 PROTOTYPE 500m RANGE SONIC ARRAY
개요

DPS(Defense Power System)는 양자 기반 자율 방어 드론으로, 도시·시설 상공에서 적 미사일·드론을 음파로 무력화합니다. 공격 기능 없음 — 방어·구조 전용. 기존 아이언돔(성공률 65~85%)보다 정밀한 근거리 방어가 목표입니다.

크기: 3.5m × 1.8m, 무게 1.5톤. 6대 동시 배치 가능. 반경 500m 내 위협 무력화.

주요 설계 특징
  • 본체 육각형 카본 파이버. 매트 블랙 + 올리브 그린 위장 패턴. 외벽 미세 회로 패턴 — "손으로 다듬은" 세련된 느낌. 부드러운 곡선 베벨 엣지.
  • 착지 시스템 AI 자율 제어 (좌우·상하 스러스터 미세 조정). 투하 후 6개 폴딩 랜딩 기어 자동 전개 → 2초 내 안정 착륙.
  • 음파 방어 모듈 육각형 스피커 패널 2개 포개어 배치. 고주파 교란으로 미사일·드론 전자 시스템 무력화. 에너지 효율 90% 이상. 반경 500m 내 위협 중화. ※ 대인 살상 기능 없음
  • 배치 방식 방어 구역 지정 배치 또는 수송기 투하. 6대 동시 운용으로 광역 방어망 구성.
기술 레퍼런스
  • 착지 기술 SpaceX 재사용 로켓 기술 + 군용 드론 GPS/레이더 연동.
  • 음파 방어 미국 LRAD(장거리 음향 장치) 기반 + 미래형 방어 증폭기 확장. 방어 전용 주파수 대역만 사용.
  • 자율 AI 2026년 기준 자율 비행 알고리즘(DJI Matrice 계열) 방어용 발전형.
