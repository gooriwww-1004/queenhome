레이센 세이

외국어 말하기 연습 — 네 목소리로 바로 피드백 받는 앱

1. 앱 개요

▼

사용자가 말하면 실시간으로 TTS/STT가 분석 → 발음·억양·속도 피드백. 언어팩 10개 이상 지원, 16:9 풀스크린 최적화.

혼자서도 "진짜 대화"처럼 연습 — AI가 상대역.

2. 주요 기능

▼
  • 음성 녹음 & 즉시 피드백: 발음 점수, 억양 그래프
  • 언어팩 선택: 영어·일본어·스페인어·프랑스어 등
  • 대화 시뮬레이션: AI가 질문 → 답변 → 교정
  • 반복 연습 모드: 틀린 문장만 자동 재생
  • 화면 비율 최적화: 태블릿·폰 모두 16:9 풀스크린

3. 설치 & 실행

▼
  1. 앱 스토어에서 "레이센 세이" 검색·설치
  2. 첫 실행 시 언어팩 다운로드 (1~2분)
  3. 마이크 권한 허용
  4. 프로필 설정 후 바로 시작

4. 규약 준수

▼

개인 학습용만 사용 — 상업 배포 금지

음성 데이터 저장 안 함, 모두 로컬 처리

공유는 스크린샷만 OK
