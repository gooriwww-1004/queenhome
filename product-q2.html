하이브리드 관제 센터 V1.7

주민 사장님 전용 — 코인원 자동 사냥 봇

1. 시스템 개요

▼

코인원 API + Tkinter GUI 자동 트레이딩 봇. 실매매/시뮬 토글 가능, 전략 ini 파일로 커스텀, 시간대별(한국·미국) 스캔, 익절·손절 자동.

전광판 + 음성 안내 + 로그 실시간 표시 — "사장님, 지금 올라요!" 느낌 그대로.

이 봇은 **API 키 없이도** 완벽히 가동돼요. REAL TRADE 체크박스만 해제하면 **가상 매매**로 실시간 시세 따라가면서 매수·매도 시뮬레이션만 돌려볼 수 있어요. 실제 돈 안 쓰고도 "아, 이 전략 괜찮네?" 테스트 가능합니다.

핵심: API 없어도 내 스타일 만들기 strategy_config.ini 파일만 열어서 — 진입 조건(RISE_RATE >= 2), 매수 비율(98% 올인), 손절(-1.5%), 익절(3%) 원하는 대로 바꾸면 돼요. 코딩 몰라도 ini 수정 한 줄로 "나만의 트레이딩 로직" 완성. MACRO, SCALPING, DEFENCE 프리셋도 다 ini로 조합 가능 — 자유롭게 튜닝하세요.

실제 매매는 API 키 넣고 REAL TRADE 체크만 하면 바로 전환. 하지만 처음엔 가상으로 충분히 놀아보세요. 안전하게, 재미있게.

2. 주요 기능

▼
  • 실매매 토글 (REAL TRADE 체크)
  • 전략 4개 프리셋 (MACRO, SCALPING, DEFENCE, CUSTOM)
  • 시장가 매수/전량 매도 (98%, 50%, 고정금액)
  • 시간대 인지: 한국 오전·오후, 미국 초반 3~20초 주기 스캔
  • 익절/손절 자동 (기본 3% / -1.5%, 전략 우선)
  • 보유코인 드롭다운 + 수동 전량 매도
  • AI 명령 하달 (텍스트 입력 → 음성 응답)
  • 잔고·로그·전광판 실시간 업데이트

3. 설치 & 실행

▼
  1. Python 3.8+ 설치
  2. pip install requests python-dotenv tkinter winsound (윈도우)
  3. .ENV 파일에 COINONE_ACCESS_TOKEN, SECRET_KEY 입력
  4. strategy_config.ini 수정 (기본 템플릿 있음)
  5. python trading_bot.py 실행

실매매 전에 API 테스트 버튼 먼저 눌러봐!

4. 주의사항

▼

실매매는 본인 책임. API 키 유출 금지.

시장가 주문 → 슬리피지 주의 (특히 저유동성 코인).

코인원 KYC 인증 필수. 잔고 조회 실패 시 .ENV 재확인.

전략 ini 수정 후 프로그램 재시작 or 재로딩 필요.

5. 장점

▼

GUI + 음성 + 전광판으로 "사장님 관제실" 느낌 그대로.

ini 파일로 전략 자유 변경 — 코딩 없이 튜닝 가능.

시뮬 → 실매매 전환 쉬움. 시간대 자동 조절로 배터리·CPU 부담 ↓
