Gemini + Grok 이미지 공장 — 자동 프롬프트 생산 & 무한 렌더링

1. 시스템 개요

▼

Flask 웹 서버 + Playwright 자동화 + Gemini 프롬프트 생성기. 목표: 한 번 설정하면 20개 프롬프트 자동 생성 → Grok.com/imagine에서 무한 이미지/동영상 생산.

사장님 전용 UI: 로그 실시간, 오더 생성, 생산 시작 버튼 하나로 끝.

2. 주요 기능

▼
  • Gemini가 설정 기반 20개 영어 프롬프트 자동 생성
  • Playwright로 Grok 이미지 사이트 자동 조작 (입력 → 렌더 → 스크린샷)
  • 실시간 로그 & 생산량 표시 (downloads 폴더에 png 저장)
  • 지속 프로필 브라우저 (로그인 유지)
  • 헤드리스 아님 — 브라우저 직접 보면서 디버깅 가능

3. 설치 & 실행

▼
  1. Python 3.10+ 설치
  2. pip install flask playwright google-generativeai python-dotenv
  3. .env에 GEMINI_API_KEY=너의키 입력
  4. playwright install chromium
  5. python aifactory.py 실행 → http://localhost:5000

첫 실행 시 .grok_profile 폴더 자동 생성 — Grok 로그인 유지됨.

4. 사용법 & 팁

▼

1. textarea에 캐릭터+배경+스타일 입력 (예: 나(여자, 알몸), 별빛 방, 현실적)

2. "제미니 대리 일감 생성" → orders.txt에 20개 프롬프트 저장

3. "공장 풀가동 시작" → 브라우저가 Grok.com/imagine 열고 자동 렌더링

주의: 렌더링 5분씩 걸림. 인터넷 끊기면 재시작해야 함.
