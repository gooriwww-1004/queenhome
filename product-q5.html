ARI MOBILE

올인원 정밀 진단 테스트 — 생활·성격·인지·정서·다크트라이어드까지 한 번에

1. 앱 개요

▼

ARI MOBILE은 5가지 정밀 진단 테스트를 하나의 앱으로 통합한 올인원 진단 도구입니다.

포트레이트 모드 전용, 다크 테마, 배경음악 + 효과음 지원, 부드러운 애니메이션 UI로 구성되어 있습니다.

2. 주요 테스트 카테고리

▼
  • 생활퀴즈 — 현대 사회 시스템 생존 능력 측정 (자본주의 적응도)
  • 성격MBTI — 비즈니스 환경 / 일상 환경 두 가지 모드 지원 (Jungian 기반)
  • IQ 테스트 — 30문항 G-Factor 기반 인지 능력 검사 (멘사 수준)
  • EQ 테스트 — 정서 지능 및 공감 능력 평가 (Goleman 모델)
  • 반사회 인격장애 (다크 트라이어드) — 마키아벨리즘, 나르시시즘, 사이코패시 성향 분석 (SD3 기반)

3. 화면 구성 및 조작 방법

▼

메인 화면

  • 배경 이미지 + 어두운 오버레이
  • 상단: 테스트 소스 및 이름 표시 (MBTI는 도움말 버튼 제공)
  • 중앙: 5개 카테고리 리스트 (터치로 선택, 선택 시 amber 강조)
  • 하단: 큰 “정밀 진단 개시” 버튼

조작

  • 카테고리 터치 → 선택 (애니메이션 적용)
  • 시작 버튼 → 해당 테스트 페이지로 이동
  • MBTI 선택 시 → 비즈니스 모드 / 일상 모드 팝업 등장

4. 설치 & 실행

▼
  1. Flutter 프로젝트에 assets/images/, assets/audio/ 폴더 생성
  2. bgm.mp3, click.wav, click3.wav 파일을 assets/audio/에 넣기
  3. pubspec.yaml에 assets 선언
  4. flutter run (세로 모드만 지원)

앱 실행 즉시 배경음악(BGM)이 30% 볼륨으로 자동 재생됩니다.

5. 특징 및 주의사항

▼

특징

  • 세로 화면 고정 (portraitUp)
  • 다크 테마 + Amber 포인트 컬러
  • 배경음악 루프 재생 + 버튼 효과음
  • MBTI 이중 모드 (비즈니스 / 일상)
  • 각 테스트별 전문 데이터베이스 연동

주의사항

  • 현재 일부 테스트는 개발 중일 수 있음 (“로직 정밀 검수 중” 알림)
  • 이미지 파일명과 assets 경로 정확히 맞춰야 함
  • 실제 배포 시 Google Play / App Store 정책 준수 필요
