CAPITAL-V v12.0 ROYAL COMMAND

앙상블 AI 실전 회의실 — 여러 뇌가 토론하는 미래

1. 시스템 개요

▼

메인 컴퓨터(태블릿/본체)에서 질문 입력 → 네트워크로 각 AI PC에 브로드캐스트 → Grok, Claude, Perplexity, Gemini가 답변 → 메인에서 모아서 순서대로 TTS 발표.

하나의 뇌가 아니라 "여러 뇌가 모여 토론하는 회의실" — 실시간 분산 앙상블.

2. 주요 기능

▼
  • 자동 질문 배포: 한 번 입력 → 모든 에이전트 동시 전송
  • 답변 수집 & 브리핑: 순서대로 목소리 바꿔 발표 (Aoede, Fenrir, Charon, Kore)
  • MASTER 요약: Gemini가 2~3문장 최종 정리
  • 실시간 상태: ONLINE/OFFLINE + 아바타 동영상
  • TTS 제어: 목소리 선택·테스트·정지·재개

3. 설치 & 실행

▼
  1. Python 3.12+ 설치
  2. pip install PyQt6 google-genai python-dotenv
  3. Genai_API.env에 API_KEY 입력
  4. avatars 폴더에 talk.mp4, idle.mp4 넣기
  5. AGENT_IPS에 PC IP (포트 5001) 설정
  6. python aimeet.py 실행

4. 규약 준수

▼

자동 구독/결제 없음 — API 키 직접 입력

배포 금지: 개인·연구용만

공유 시 텍스트·스크린샷·캡처만 OK

5. 장점

▼
